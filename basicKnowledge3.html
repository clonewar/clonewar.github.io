Javaの基礎知識(3)

開発環境について

Java Development Kit(JDK)

Javaでプログラムを作成する際に必要な開発キット

参考文献：http://ja.wikipedia.org/wiki/Java_Development_Kit

Java Runtime Environment(JRE)

Javaで作成されたプログラムを実行する際に必要な実行環境
　・PrivateJRE(非公開JRE)：JDKに含まれているJRE
　・Public JRE( 公開JRE)：サイト等から単体でインストールしたJRE

参考文献：http://ja.wikipedia.org/wiki/Java_Runtime_Environment

Java Platform, Standard Edition (JavaSE)

標準的な機能が纏められたAPIの集合体
※バージョン5.0まではJava 2 Platform,Standard Edition(J2SE)と呼ばれていた

API
基本言語ライブラリー,通信ライブラリー,GUI関連ライブラリー
JDBC,JNDI,RMI,Java IDL(CORBA),JMX

参考文献：http://ja.wikipedia.org/wiki/Java_Platform,_Standard_Edition

Java Platform, Enterprise Edition (JavaEE)

Javaの企業用機能セット
JavaSEの拡張機能の形で提供される
(JavaSE + JavaEE用ライブラリ + 標準JavaEEサーバ + 各種ドキュメント)
※バージョン5.0まではJava 2 Platform,Enterprise Edition(J2EE)と呼ばれていた

API
サーブレット,JSP,JSF,EJB,JPA,JMS,JTA,JCA
Webサービス関連ライブラリー,Java Mail,DIコンテナ関連

参考文献：http://ja.wikipedia.org/wiki/Java_Platform,_Enterprise_Edition

Application Programming Interface(API)

Javaでは関数群のことを指します
クラスやインタフェースの形で提供されています
APIは数多く存在する為、使い方は公式サイトで調べます

参考文献：http://docs.oracle.com/javase/jp/8/api/

Java Virtual Machine(JavaVM,JVM)

Java言語で作られたプログラムを実行するためのソフトウェアです
JREに含まれています
実行させるにはソースコードをコンパイルしてクラスファイルを作成する必要がある

参考文献：http://ja.wikipedia.org/wiki/Java仮想マシン
